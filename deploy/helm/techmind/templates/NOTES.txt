TechMind has been deployed!

Release Name: {{ .Release.Name }}
Namespace: {{ .Release.Namespace }}

1. Check the server status:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/component=server

2. Check the worker status:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/component=worker

3. Get the server service URL:
   export POD_NAME=$(kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/component=server -o jsonpath="{.items[0].metadata.name}")
   kubectl port-forward -n {{ .Release.Namespace }} $POD_NAME 8080:8080

4. Check Prometheus metrics:
   curl http://localhost:8080/metrics
